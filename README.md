# imgToVector 🎨

Convierte imágenes rasterizadas a vectores SVG de alta calidad.

## Características
- Soporta múltiples formatos (PNG, JPG, WEBP, etc.)
- Múltiples presets de vectorización
- Descarga automática en SVG o ZIP
- Hecho con FastAPI y vtracer
